Software, Code, and Data

The software and code that is used in our influenza diallel paper can be found at the following GitHub website:

flu-diallel repository

The corresponding R package, called "treatmentResponseDiallel", can be used to reproduce analyses in the influenza dialllel paper. It can be installed via instructions at the following link:

treatmentResponseDiallel

In order to collect some of the functions that I have written that are useful for working with Collaborative Cross data and various other analyses, I made the following R package:

PLMcctools

The following R package contains functions and data that help reproduce analyses in the litter diallel manuscript, now in preparation:

litterDiallel

The R code to reproduce analyses in the malaria infant study manuscript, now in preparation, will be housed here:

malariaInfantStudy
